
### Join entre os resultados dos anos
```sql
SELECT resul_1995.*, resul_2010.NOME_2010
FROM resul_1995, resul_2010
WHERE resul_1995.[NUMERO DO CONTRIBUINTE] = resul_2010.[NUMERO DO CONTRIBUINTE];
```
